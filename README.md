# 💰 FinTrack

A modern full-stack personal finance & expense tracking application.

> 🚧 Currently under development.

## ✨ Features

- 🔐 JWT Authentication
- 💵 Income Management (CRUD)
- 💸 Expense Management (CRUD + Search & Filter)
- 🎯 Budget Planning
- 📊 Responsive Dashboard
- 🌙 Dark / Light Mode

## 🛠 Tech Stack

- Next.js
- React
- Tailwind CSS
- Node.js
- Express.js
- MongoDB
